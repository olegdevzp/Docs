## üìù Notes

- **Sequential Shortcuts**: Some shortcuts like `Cmd + K, Cmd + W` are sequential - press `Cmd + K`, release, then press `Cmd + W`
- **Context Matters**: Many shortcuts work differently based on context (editor, terminal, sidebar)
- **Custom Shortcuts**: You can customize any shortcut via `Cmd + K, Cmd + S` (Keyboard Shortcuts)
- **AI Context**: Always select relevant code before using `Cmd + K` for better AI suggestions
- **Practice Makes Perfect**: Focus on mastering 5-10 shortcuts at a time
